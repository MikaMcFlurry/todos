# Finanztracker 48h Todo & Timeline

Statisches HTML-Tool für die nächsten 48h im Finanztracker-Projekt.

## Nutzung

Die Seite liegt in `index.html` und benötigt keine API, keine JSON-Datei und keine externen Abhängigkeiten.

Funktionen:

- Todos abhaken
- Fortschritt visuell sehen
- Timeline-Status sehen
- Notizen speichern
- Fortschritt als JSON exportieren/importieren

Der Fortschritt wird im jeweiligen Browser per `localStorage` gespeichert. Für Gerätewechsel oder als Backup bitte den Export-Button nutzen.

## GitHub Pages

Wenn GitHub Pages für dieses Repository aktiviert ist, ist die Seite typischerweise hier erreichbar:

`https://mikamcflurry.github.io/todos/`

Falls die Seite nicht erreichbar ist:

1. GitHub Repo öffnen: `MikaMcFlurry/todos`
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main`
5. Folder: `/ (root)`
6. Save

Danach einige Minuten warten.
